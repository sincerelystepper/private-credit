Key Projects

Cognitive Radio Listener & Helical Antenna

Architected a full-stack embedded wireless system to combat spectrum congestion, achieving a 59% increase in Packet Delivery Ratio.

Designed a custom circularly polarized helical antenna with 10.8 dBi gain using ANSYS HFSS. Developed bare-metal C/C++ firmware on a Raspberry Pi Pico implementing a custom Cognitive MAC protocol.

Robust Digital Servo Motor Control

Engineered a cascaded digital PID control system for precise servo motor regulation under dynamic load.

Designed and deployed a Delta PLC-based HMI with safety interlocks. Performed system identification using MATLAB/Simulink to model plant dynamics and optimize controller design.

AI for Power - Predictive Maintenance

Spearheaded the design of a three-phase voltage/current sensing system for predictive maintenance of electrical infrastructure.

Architected the data acquisition front-end with high-precision ADCs for harmonic analysis. Built the prototype on a Raspberry Pi, establishing a hardware pipeline for cloud-based time-series data.

Autonomous Embedded Wildlife Monitor

Developed a low-power, autonomous system using ESP32 to monitor African penguin chick growth.

Integrated sensors (temperature, humidity, load cell) and ESP32-CAM. Implemented fault-tolerant data acquisition with time-stamped logging for wildlife research.

FPGA-Based Image Filtering (FIFA-ED)

Designed an FPGA-accelerated image edge detection system for real-time vision.

Implemented Sobel and Canny filters in Verilog on Xilinx Vivado, optimizing a multi-stage pipeline for resource efficiency.
